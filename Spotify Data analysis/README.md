# Spotify Music Data Analysis using Microsoft Excel

## Project Overview

This project analyzes Spotify music data using Microsoft Excel to uncover trends in song popularity, artist performance, audio features, and genre distribution. The analysis demonstrates how Excel can be used for data cleaning, exploratory data analysis (EDA), and dashboard creation to transform raw music data into actionable insights.

---

# Project Objectives

The main objectives of this project were to:

* Clean and prepare the Spotify dataset for analysis.
* Explore patterns in music popularity.
* Analyze relationships between audio features.
* Identify top-performing artists and genres.
* Create interactive visualizations and dashboards.
* Present insights in a clear and user-friendly format.

---

# Dataset

The dataset contains Spotify track information, including attributes such as:

* Track Name

* Popularity Score
* Danceability

* Instrumentalness
* Loudness
* Speechiness
* Tempo
* Valence
* Duration
* Key
* Mode
* Time Signature

# Data Cleaning

The dataset was cleaned by:

* Checking for missing values.
* Removing duplicate records where necessary.
* Standardizing formatting.
* Converting song duration into a more readable format.
* Preparing the dataset for analysis and visualization.

---

# Exploratory Data Analysis (EDA)

The analysis focused on:

* Distribution of song popularity.
* Top artists based on popularity.
* Genre analysis.
* Audio feature comparisons.
* Relationships between popularity and song characteristics.
* Summary statistics using pivot tables.

---

# Dashboard Features

The interactive Excel dashboard includes:

* Popularity analysis.
* Artist comparisons.
* Genre distribution.
* Interactive slicers for filtering data.
* Charts highlighting key music trends.

---

# Key Insights

Some of the insights obtained from the analysis include:

* Certain artists consistently have higher popularity scores than others.
* Music genres differ in their average popularity and audio characteristics.
* Highly danceable and energetic songs tend to appear frequently among popular tracks.
* Interactive filtering enables deeper exploration of different artist and genre trends.
